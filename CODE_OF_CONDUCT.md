# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant], a
widely-used code of conduct for open-source communities.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Our pledge

We — contributors and maintainers — pledge to make participation in this
project a harassment-free experience for everyone, regardless of age,
body size, visible or invisible disability, ethnicity, sex
characteristics, gender identity and expression, level of experience,
education, socio-economic status, nationality, personal appearance,
race, caste, colour, religion, or sexual identity and orientation.

We pledge to act and interact in ways that contribute to an open,
welcoming, diverse, inclusive, and healthy community.

## Scope

This Code of Conduct applies within all project spaces — the GitHub
repository, issue tracker, pull-request discussions, project
communications — and also when an individual is officially representing
the project in public spaces (e.g., using an official project email
address, posting via an official social-media account, or acting as a
designated representative at an online or offline event).

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behaviour
may be reported to the project maintainers by opening a GitHub issue
marked **private** via the `Security` tab, or by contacting the
maintainers directly via the addresses listed on the project's GitHub
page.

All complaints will be reviewed and investigated promptly and fairly.
Maintainers are obligated to respect the privacy and security of the
reporter of any incident.

## Enforcement guidelines

Maintainers will follow the [Contributor Covenant Enforcement
Guidelines][enforcement] when determining the consequences for any
action they deem in violation of this Code of Conduct. The short
summary is: correction → warning → temporary ban → permanent ban,
proportional to severity and recurrence.

[enforcement]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
